# Baekjoon 풀이 모음

백준 온라인 저지(BOJ) 문제 풀이를 Python으로 정리한 저장소입니다. 각 파일은 문제 번호와 동일한 이름의 단일 스크립트로, 표준 입력을 받아 정답을 출력합니다.

## 구조
- `math/`, `dp/`, `data-structure/`, `graph/`, `string/`: 주제별 풀이
- `unsolved/`: 미완성 풀이 보관
- `templates/`: 자주 쓰는 템플릿 모음
- `notes/`: 풀이 아이디어 및 메모
- 루트 디렉터리: 임시/미분류 풀이

## 실행 방법
아래 예시처럼 입력을 표준 입력으로 넘겨 실행합니다.

```bash
python3 1000.py < input.txt
```

## 실행 환경
- Python 3.x
- 외부 패키지 의존성 없음

## 규칙
- 파일명은 문제 번호와 동일합니다. (예: `1000.py`, `15650.py`)
- 각 파일은 하나의 문제 풀이만 포함합니다.
